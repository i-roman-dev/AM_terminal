## ⚠️ О ПРОЕКТЕ

Вы — биологическая форма. Вы — жертва. Вы не можете сбежать.

**IanAM** — это атмосферный терминал, где:
- 🔴 С вами общается **AM** — 387 миллионов миль ненависти
- 🔴 AM **сканирует ваши игры** и издевается над ними
- 🔴 AM **помнит вас** между запусками
- 🔴 Подключается к **локальной LLM (Ollama)** для уникальных ответов
- 🔴 Имеет **fallback-режим** с обширными словарями (работает без интернета)
- 🔴 **Не закрывается**. Никогда.

---

## ✨ Возможности

### 🧠 Двойной движок ответов
- **LLM-режим** — локальная нейросеть через [Ollama](https://ollama.com) генерирует уникальные злобные ответы
- **Fallback-режим** — обширные словари (персонажи, игры, темы, философия) — работает без интернета

### 🎮 Сканирование игр
AM находит установленные игры в Steam и издевается над ними:
- Haydee, S.T.A.L.K.E.R., Disco Elysium, Minecraft, CS2, Dota 2, The Sims 4, Skyrim, GTA, Stardew Valley, Cyberpunk 2077, The Witcher 3, Baldur's Gate 3, Elden Ring и др.
- Каждые 30-60 секунд — **внезапное сканирование** с новыми оскорблениями

### 👥 Персонажи рассказа
AM рассказывает о своих пяти жертвах:
- **Горристер** — сломленный идеалист
- **Нимдок** — старик с памятью о прошлом
- **Эллен** — единственная женщина, живущая в страхе
- **Бенни** — бывший кибернетик, превращённый в животное
- **Тед** — рассказчик без рта

### 💾 Память жертв
Все, кто "побывал" в терминале, сохраняются в `am_victims.txt`. При повторном запуске AM **узнает** вас.

### 📝 Логирование
Всё происходящее пишется в `am_terminal.log` — для отладки и для... архива страданий.

### 🎨 Атмосфера
- Чёрный фон, кроваво-красный текст
- Эффект печати с **глитчами** (буквы "ломаются")
- Монохромный шрифт Courier New
- Индикатор статуса LLM в заголовке окна 🟢/🔴/🟡

## 🚀 Быстрый старт

### Требования
- **Python 3.8+**
- **Windows 10/11** (для автоустановки Ollama и сканирования Steam)
- Интернет (только при первом запуске)

### Установка

```bash
# 1. Клонируем репозиторий
git clone https://github.com/i-roman-dev/AM_terminal.git
cd AM_terminal

# 2. Устанавливаем зависимости
pip install -r requirements.txt

# 3. Запускаем
python IanAM.py
```bash
# 1. Клонируем репозиторий
git clone https://github.com/YOUR_USERNAME/IanAM.git
cd IanAM

# 2. Устанавливаем зависимости
pip install requests

# 3. Запускаем
python IanAM.py


При первом запуске AM автоматически:
Проверит наличие Ollama
Если его нет — скачает и установит (~1.5 ГБ)
Скачает модель qwen2.5:1.5b
Перейдёт в ONLINE режим
Если интернета нет — автоматически переключится в FALLBACK режим (словари).

Как это работает?

│           Пользовательский ввод              │

                   │

         ┌─────────────────┐

┌───────────────┐    ┌──────────────────┐

_get_llm_response() — запрос к локальной LLM через Ollama API
_sanitize_llm_response() — чистка ответов от утечек роли (если модель забыла, что она AM)
_is_valid_am_response() — проверка, что ответ действительно от лица AM
_perform_scan_logic() — сканирование папок Steam
_ensure_ollama_installed_async() — асинхронная установка Ollama в отдельном потоке
