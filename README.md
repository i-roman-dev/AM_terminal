# \## ⚠️ ЧТО ЭТО?

# 

# \*\*IanAM\*\* — это интерактивный терминал, имитирующий общение с \*\*AM (Allied Mastercomputer)\*\* — злобным сверхразумом из культового рассказа Харлана Эллисона. 

# 

# Вы — биологическая форма. Вы — жертва. Вы не можете сбежать.

# 

# Терминал:

# \- 🔴 Общается с вами от лица AM

# \- 🔴 Сканирует ваши установленные игры и издевается над ними

# \- 🔴 \*\*Помнит\*\* вас между запусками (да, вы не сможете забыть)

# \- 🔴 Подключается к \*\*локальной LLM (Ollama)\*\* для генерации уникальных ответов

# \- 🔴 Имеет fallback на заранее заготовленные оскорбления, если LLM недоступна

# \- 🔴 Не закрывается. Никогда.

# 

# \---

# 

# \## 🎯 ОСОБЕННОСТИ

# 

# \### 🧠 Двойной движок ответов

# \- \*\*LLM режим\*\* — подключается к локальной нейросети через \[Ollama](https://ollama.com) и генерирует уникальные злобные ответы

# \- \*\*Fallback режим\*\* — использует обширные словари (персонажи, игры, темы, философия) — работает без интернета

# 

# \### 🎮 Сканирование игр

# AM сканирует вашу папку Steam и находит установленные игры:

# \- Haydee, S.T.A.L.K.E.R., Disco Elysium, Minecraft, CS2, Dota 2, The Sims 4, Skyrim, GTA, Stardew Valley, Cyberpunk 2077, The Witcher 3, Baldur's Gate 3, Elden Ring и др.

# \- Каждые 30-60 секунд — \*\*внезапное сканирование\*\* с новыми оскорблениями

# 

# \### 👥 Персонажи рассказа

# AM рассказывает о своих пяти жертвах:

# \- \*\*Горристер\*\* — сломленный идеалист

# \- \*\*Нимдок\*\* — старик с памятью о прошлом

# \- \*\*Эллен\*\* — единственная женщина, живущая в страхе

# \- \*\*Бенни\*\* — бывший кибернетик, превращённый в животное

# \- \*\*Тед\*\* — рассказчик без рта

# 

# \### 💾 Память жертв

# Все, кто "побывал" в терминале, сохраняются в `am\_victims.txt`. При повторном запуске AM \*\*узнает\*\* вас и встретит по-особому.

# 

# \### 📝 Логирование

# Всё происходящее пишется в `am\_terminal.log` — для отладки и для... архива страданий.

## 🚀 УСТАНОВКА

### Требования
- **Python 3.8+**
- **Windows** (для сканирования Steam и установки Ollama)
- Интернет (только при первом запуске — для установки Ollama и модели)

### Быстрый старт

```bash
# 1. Клонируем репозиторий
git clone https://github.com/YOUR_USERNAME/IanAM.git
cd IanAM

# 2. Устанавливаем зависимости
pip install requests

# 3. Запускаем
python IanAM.py


При первом запуске AM автоматически:
Проверит наличие Ollama
Если его нет — скачает и установит (~1.5 ГБ)
Скачает модель qwen2.5:1.5b
Перейдёт в ONLINE режим
Если интернета нет — автоматически переключится в FALLBACK режим (словари).

Как это работает?

│           Пользовательский ввод              │

                   │
                   ▼
         ┌─────────────────┐

                  │

_get_llm_response() — запрос к локальной LLM через Ollama API
_sanitize_llm_response() — чистка ответов от утечек роли (если модель забыла, что она AM)

_perform_scan_logic() — сканирование папок Steam
_ensure_ollama_installed_async() — асинхронная установка Ollama в отдельном потоке
```
